# 🐺 狼人杀游戏完整流程详细说明

## 📱 界面设计理念

### 视觉风格
- **主色调**：渐变紫蓝色背景（#667eea → #764ba2）
- **玻璃拟态**：半透明卡片效果，营造现代感
- **移动端优先**：所有按钮最小44px，输入框16px字体
- **响应式布局**：完美适配手机、平板、桌面端

### 交互原则
- **简化操作**：每页核心功能不超过3个
- **即时反馈**：所有操作都有视觉响应
- **防误触**：重要操作需要二次确认
- **无障碍**：支持键盘导航和屏幕阅读器

##🌙 第五步：夜晚流程与法官系统

### 5.1 夜晚阶段整体流程
**法官喊人睁眼的标准流程：**
```
第1夜（特殊流程）：
├─ 丘比特睁眼 → 连接情侣
├─ 情侣睁眼 → 互相确认身份
├─ 守卫睁眼 → 选择守护目标
├─ 狼人睁眼 → 互相确认身份 → 选择击杀目标
├─ 女巫睁眼 → 查看击杀信息 → 决定是否使用解药/毒药
└─ 预言家睁眼 → 选择查验目标

第2夜及以后：
├─ 守卫睁眼 → 选择守护目标
├─ 狼人睁眼 → 选择击杀目标
├─ 女巫睁眼 → 查看击杀信息 → 决定是否使用解药/毒药
└─ 预言家睁眼 → 选择查验目标
```

### 5.2 法官喊人睁眼的具体话术
**标准法官台词：**
```
🌙 "天黑了，请所有玩家闭眼"

⏰ "丘比特请睁眼，请选择你要连接的两位情侣"
⏰ "情侣请睁眼，互相确认身份"（仅第1夜）

⏰ "守卫请睁眼，请选择你要守护的玩家"

⏰ "狼人请睁眼，狼人互相确认身份，请选择你们要击杀的目标"

⏰ "女巫请睁眼，昨晚死亡的是[玩家姓名]，你要使用解药吗？你要使用毒药吗？"

⏰ "预言家请睁眼，请选择你要查验身份的玩家"
```

### 5.3 防作弊机制详解
**睁眼时机控制：**
```
防作弊规则：
├─ 只有在法官喊到对应角色时才能睁眼
├─ 提前睁眼会被系统检测并记录
├─ 错误时机睁眼会在聊天框显示警告
└─ 多次违规会被标记为可疑玩家
```

**违规检测系统：**
```
聊天框警告显示（红色大号字体）：
⚠️ "某某玩家尝试睁眼，已被拦截"

违规等级：
├─ 第1次：警告提示
├─ 第2次：黄色标记
├─ 第3次：红色标记
└─ 第4次：移出游戏房间
```

**移动端特殊处理：**
```
手机端显示差异：
├─ 警告文字正常大小（避免遮挡）
├─ 使用震动反馈代替视觉警告
├─ 简化提示信息
└─ 增加确认按钮防止误触
```

