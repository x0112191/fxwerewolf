狼人杀身份验证系统
项目简介
狼人杀身份验证系统是一款基于前端技术构建的轻量级身份管理工具，专为狼人杀游戏场景设计，提供用户密钥创建、身份验证、管理员审核等核心功能。系统采用现代化的 UI 设计（玻璃拟态 + 渐变风格），全端适配移动端 / 桌面端，操作流程简洁直观，可快速集成到狼人杀游戏场景中。
核心功能
1. 用户端功能
密钥创建：用户填写身份标识姓名，设置 6 位以上专属密钥（数字 / 字母组合）
身份验证：通过密钥登录系统，仅审核通过的用户可进入游戏界面
密码可视化：支持密码显示 / 隐藏切换，提升输入体验
操作反馈：所有操作均有 Toast 提示，替代传统 alert 弹窗，交互更友好
2. 管理端功能
管理员验证：通过专属密钥进入审核后台（支持多密钥：A /admin666 / 123456）
用户审核：查看待审核 / 已审核用户列表，支持通过 / 拒绝审核操作
审核管理：支持批量处理用户审核，空列表友好提示，操作记录实时反馈
安全退出：管理员可随时退出审核模式，返回登录页面
3. 技术特性
响应式设计：全尺寸适配手机、平板、桌面端，触摸区域最小 44px 符合移动端规范
视觉风格：渐变紫蓝色背景 + 玻璃拟态卡片，现代感 UI 设计
本地存储：基于 localStorage 实现用户数据持久化，无需后端快速验证
服务器关联：支持绑定服务器地址（默认：111.666.666.666），便于后续对接真实服务端
快速开始
运行环境
任意现代浏览器（Chrome/Firefox/Safari/Edge）
无需后端依赖，纯前端运行
运行步骤
将代码保存为index.html文件
直接用浏览器打开该文件即可启动系统
可选：部署到任意静态服务器（Nginx/Apache/Netlify 等）
基础操作流程
普通用户
打开页面，点击「创建密钥」
填写 1-10 位姓名，设置 6 位以上密钥
确认提交后等待管理员审核
审核通过后，返回登录页输入密钥即可登录
管理员
打开页面，点击「更多选项」→「管理员入口」
输入管理员密钥（0）
进入审核页面，处理待审核用户（通过 / 拒绝）
完成操作后点击「退出管理员」返回登录页

自定义配置
1. 修改服务器地址
找到代码中以下常量，替换为实际服务器地址：
javascript
运行
const SERVER_URL = '111.666.666.666'; // 替换为真实服务器IP/域名
2. 新增 / 修改管理员密钥
修改handleAdminLogin函数中的adminKeys数组：
javascript
运行
const adminKeys = [0]; // 新增/删除管理员密钥
3. 调整视觉样式
渐变背景：修改.gradient-bg中的linear-gradient值
玻璃拟态：调整.glass-effect中的backdrop-filter/background值
提示框样式：修改.toast中的背景色 / 位置 / 过渡效果
待扩展功能（可选）
对接后端接口：将 localStorage 替换为 fetch/axios 请求，连接真实数据库
增加密码强度检测：密钥创建时检测复杂度（大小写 / 特殊字符）
审核日志记录：保存管理员操作记录，支持查看 / 导出
批量审核功能：管理员可批量通过 / 拒绝用户审核
密钥找回功能：添加邮箱 / 手机号验证，支持密钥重置
注意事项
本地存储数据仅保存在浏览器中，清空缓存会丢失数据（生产环境需对接后端）
管理员密钥建议部署前修改为强密钥，避免安全风险
姓名 / 密钥提交后不可修改，创建时需提示用户确认
移动端使用时建议开启横屏模式，提升操作体验
技术栈
HTML5：语义化标签 / 对话框组件
TailwindCSS：原子化 CSS 框架，快速实现响应式设计
DaisyUI：UI 组件库，提供按钮 / 表单 / 弹窗等基础组件
JavaScript：原生 JS 实现逻辑，无框架依赖
Font Awesome：图标库，提升视觉体验
